Why?

Well, you got all the resources and links, but why would you use PopifyJS?

Easy to Use

This library is very easy to use as it does not use any complex functions and uses a function to show a popup.

Responsive

PopifyJS works on any device, with any options enabled or disabled, so you need not worry about showing it on small or large devices.

Customizable

You can customize the popup as you want through your own styles or change the structure by yourself.

No Dependencies

PopifyJS uses no heavy dependencies which slow down your website, just the lightweight bootstrap icons used for button and main popup icons.
