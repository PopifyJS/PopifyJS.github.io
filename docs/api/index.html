API Reference

If you are not aware of the starter code to use PopifyJS, go to the main docs page and from there, the quickstart page.

All options in the popify(...) function are optional. You need not write out any unecessary option if your popup does not need it.

headerContent

this is a parameter in the popify() function. It is used to set the textContent of the popup's header/title.
[String]

popify({
  headerContent: "Popup Title",
  ...
});
subText

this is a parameter in the popify() function. It is used to set the sub text of the popup.
[String]

popify({
  subText: "some brief description you want to give in the popup",
  ...
});
Overlay

this is a parameter in the popify() function. It specifies whether an overlay should be present behind the opup or not.
[Boolean]

popify({
  overlay: true,
  ...
});
closeOnBackground

This is a parameter in the popify() function and it specifies whether the popup should close when the background/overlay is clicked.
[Boolean]

popify({
  closeOnBackground: false,
  ...
});
closeOnEscape

This is the parameter in the popify() function which is used to specify whether the popup should be closed on [Escape] key press or not.
[Boolean]

popify({
  closeOnEscape: true,
  ...
});
short

This parameter in the popify() function is used to specify whether the popup's size should be large or small. [responsive in both options]
[Boolean]

popify({
  short: false,
  ...
});
icon

This parameter in the popify() function puts an icon in the popup if given.
[String - {Options}]

The options for the popup icons are:

  • Success
  • Error
  • Warning
  • Info
  • Question

popify({
  icon: "error", // should be lowercase
  ...
});
closeIcon

This parameter in the popify() function specifies if you want a close icon in the popup or not.
[Boolean]

popify({
  closeIcon: true,
  ...
});
Buttons

There is one more parameter in the popify() function. It is something like this:
[Array - {Object}]

popify({
  buttons: [
    {
      ...
    },
    {
      ...
    }
  ],
  ...
});

The buttons parameter is initially an array, which contains objects containing parameters which set the button's functions and styles. The array can contain as many button objects but you need to be careful on how it shows up in the popup.

text

This is a parameter in the object of buttons array. This sets the button's text
[String]

popify({
  buttons: [
    {
      text: "button text you want",
      ...
    }
  ],
  ...
});
closePopup

This is a parameter in the object of buttons array. This specifies whether the popup should close when the specific button is clicked or should remain open.
[Boolean]

popify({
  buttons: [
    {
      closePopup: true, // can be false
      ...
    }
  ],
  ...
});
class

This is a parameter in the object of buttons array. This gives an extra custom class to the button if specified in the parameter.
[String]

popify({
  buttons: [
    {
      "class": "custom-class-you-want-to-give-to-button" // make sure there are no spaces.
    }
  ],
  ...
});

Make sure that the "class" parameter is in quotes, because class is an existing javascript keyword and it will result in an error otherwise.

icon

This is a parameter in the object of buttons array. This sets the button's icon if specified.
[String - Bootstrap Icon]

popify({
  buttons: [
    {
      icon: "trash" // this is a valid Bootstrap icon name.
    }
  ],
  ...
});

For seeing all the available icons in Bootstrap, visit Bootstrap Icons. The naming system should be all lowercase. For example: icon: "arrow down circle"

type

This is a parameter in the object of buttons array. This specifies the button type [optional, as all the parameters]. It is basically a way to change colors of the buttons without you using any extra style.
[String - {type}]

The types are:

  • error
  • success
  • warning
  • info

popify({
  buttons: [
    {
      type: "warning" // any of the given 4 types work here. only lowercase.
    }
  ]
});
run

This is a parameter in the object of buttons array. It should be a function which runs when the button is clicked.
[Function]

popify({
  buttons: [
    {
      run: () => {
        alert("Button Clicked!"); // this runs when the button is clicked.
      }
    }
  ]
});

This can also be a variable which stores a function, rather than a function in the parameter itself.
