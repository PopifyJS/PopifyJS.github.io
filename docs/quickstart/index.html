Quickstart

Get started on putting a popup using PopifyJS here

This page only contains the starter code for using PopifyJS. For full information on the library, visit the API Reference.


Firstly, to actually import the library, you need to put the code given below in your html file:

<script src="https://popifyjs.github.io/src/script.js"></script>
<link rel="stylesheet" href="https://popifyjs.github.io/src/style.css">

Now, to make a popup, simply use the popify(...) function

popify({
  overlay: true,
  closeOnBackground: true,
  closeOnEscape: true,
  short: true,
  icon: "popup icon",
  closeIcon: true,
  headerContent: "Popup Header",
  subText: "this is the sub text of the popup",
  
  buttons: [
    {
      text: "Button 1",
      closePopup: true,
      class: "button-class",
      icon: "button icon",
      type: 'none',
      run: () => {
        console.log("First button clicked");
      }
    },
    {
      text: "Button 2",
      closePopup: true,
      class: "button2-class",
      icon: "button icon",
      type: 'none',
      run: () => {
        console.log("Second button clicked");
      }
    }
  ]
});

This code has all the current PopifyJS functions and methods. Learn more through the API Reference.
You can test all of this through the PopifyJS Playground [home page]
